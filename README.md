# 🏥 Centro Médico de Olhos - Interface Profissional

[![GitHub Pages](https://img.shields.io/badge/Deploy-GitHub%20Pages-blue?style=flat-square&logo=github)](https://wallaceleaal.github.io/Hospital/)

Este projeto é uma interface de gerenciamento para uma clínica oftalmológica, desenvolvida com foco em **UI/UX (User Interface & User Experience)** de alto nível. O objetivo foi aplicar conceitos modernos de design para transmitir a seriedade e a confiança necessárias em ambientes de saúde.

👉 **[Acesse o Site ao Vivo](https://wallaceleaal.github.io/Hospital/)**

---

## 🛠️ Tecnologias e Conceitos Aplicados

O projeto foi construído utilizando as melhores práticas de desenvolvimento Web I:

* **HTML5 Semântico:** Para melhor acessibilidade e indexação (SEO).
* **CSS3 Avançado:** * **Glassmorphism:** Uso de `backdrop-filter: blur` para criar o efeito de vidro fosco nas seções.
    * **Paleta Médica:** Escolha estratégica de cores (MidnightBlue e SteelBlue) para psicologia das cores no setor de saúde.
    * **Responsividade:** Layout adaptável para diferentes tamanhos de tela.
* **Git & GitHub:** Versionamento completo e deploy automatizado via GitHub Pages.

---

## 🚀 Funcionalidades Principais

- [x] **Cadastro de Especialistas:** Interface para registro de médicos e valores de consulta.
- [x] **Gestão de Dados:** Tabela estruturada para visualização rápida de corpo clínico.
- [x] **Triagem Visual:** Listagem intuitiva de sintomas para agilizar o atendimento.
